# Motion: Scroll-linked Animations

Motion uses browser APIs for hardware accelerated scroll-linked and scroll-linked animations.

<Visualizer Comp={ScrollLinkedAnimations} reload={false}>
```tsx showLineNumbers {3-5}
import { animate, scroll } from 'motion'

scroll(animate(".box", { rotate: [0, 360] }, { ease: "linear" }), {
  target: '#container'
})

<div id="container">
    <div class="box" style="position: sticky">
      I'm rotating as you scroll!
    </div>
</div>

```
</Visualizer>
