# Timeline

The `animate` function can also accept complex animation sequences.
A **sequence** is an array of animate definitions:

<Visualizer Comp={VariantsComp}>
```tsx showLineNumbers {4-5}
import { animate, stagger } from 'motion'

animate([
  [".box", { opacity: [0, 1], scale: [0, 1] }],
  [".item", { opacity: [0, 1], y: [20, 0] }, { delay: stagger(0.2) }],
]);

```
</Visualizer>
