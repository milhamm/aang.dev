# Transition

We can define different types of animation by changing the tranisiton type.

<Visualizer Comp={TransitionComp} reload={false}>
```tsx showLineNumbers {4,6,14,20}
import { animate } from 'motion';

// Tween transition
animate('.box', { x: [0, 200] }, { type: 'tween', ease: 'easeInOut', duration: 0.5 })
// Spring transition
animate('.box', { x: [0, 200] }, { type: 'spring', duration: 0.5, bounce: 0.2 })

// React
import { motion } from 'motion/react';
// Tween transition
<motion.div
  initial={{ x: 0 }}
  animate={{ x: 200 }}
  transition={{ type: 'tween', ease: 'easeInOut', duration: 0.5 }}
/>
// Spring transition
<motion.div
  initial={{ x: 0 }}
  animate={{ x: 200 }}
  transition={{ type: 'spring', duration: 0.5, bounce: 0.2 }}
/>
```
</Visualizer>

## Defaults

- Physical properties like `x` or `scale` will be animated via a _spring simulation_.
- Properties like `opacity` or `color` will be animated with a _tween_.
