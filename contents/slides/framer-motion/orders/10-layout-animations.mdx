# Layout Animations

Animate a `motion` component between distinct _layouts_ by setting the `layout` prop to `true`.
This will result in what we call a layout animation. For example:

<Visualizer Comp={LayoutAnimation} reload={false}>
```tsx showLineNumbers {11}
import { motion } from 'framer-motion'

<div
  style={{
    display: 'flex',
    flexDirection: isColumn ? 'column' : 'row'
  }}
>
  {items.map((item) => (
    <motion.div
      layout
      key={item}
    >
      {item}
    </motion.div>
  ))}
</div>
```
</Visualizer>

## But why not normal CSS?
1. [Not all CSS properties are animatable](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_animated_properties) e.g `flex-direction`
2. CSS layouts are difficult and [expensive to animate](https://web.dev/articles/animations-overview).
