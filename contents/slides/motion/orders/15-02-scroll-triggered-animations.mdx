# Motion React: Scroll-triggered Animations

The `whileInView` prop can be used to create scroll-triggered animations

<Visualizer Comp={ScrollTriggeredAnimations} reload={false}>
```tsx showLineNumbers {5, 10-11}
import { motion } from 'motion/react'

<motion.div
  initial={{ opacity: 0 }}
  whileInView={{ opacity: 1 }}
/>

<motion.div
  initial={{ opacity: 0, y: 40 }}
  whileInView={{ opacity: 1, y: 0 }}
  viewport={{ amount: 0.8, once: true }}
/>

```
</Visualizer>
