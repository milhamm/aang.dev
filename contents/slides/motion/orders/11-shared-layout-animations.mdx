# Motion React: Shared Layout Animations

When a new component is added that has a `layoutId` prop that matches an existing component, it will automatically animate out from the old component.

<Visualizer Comp={SharedLayoutAnimation} reload={false}>
```tsx showLineNumbers {6}
import { motion } from 'motion/react'

<ul>
  {items.map((item) => (
    <li key={item} className="relative">
      {isSelectedTabs && <motion.div layoutId="tabIndicator" />}
      <span>{item}</span>
    </li>
  ))}
</ul>
```
</Visualizer>
